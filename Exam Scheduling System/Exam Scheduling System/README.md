# Examination Scheduling System

This project is a Python-based Examination Scheduling System developed as a final year project. It automates the process of generating exam timetables, assigning invigilators, and managing scheduling constraints.

## Features
- Automated exam scheduling
- Invigilator assignment system
- Conflict detection and validation
- CSV-based data handling
- Web interface (Flask)

## Technologies Used
- Python
- Flask
- SQL / File-based storage
- HTML, CSS
- JavaScript

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open in browser:
   http://127.0.0.1:5000/
